#Example-app with three differents flavors: react v17 (classComponent and typescript),Next.js (Functional component + SSR) && Next.js (Functional component + CSR). 

Run Locally 📦

1- Clone repo 🃏

    $ git clone https://github.com/mselmank/example-app.git
 
2- ## Run Frontend  🚀     
          
        # open new terminal
        $ npm i
        $ npm run dev
